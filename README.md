# basafer
لوحة تحكم (Dashboard) احترافية ومتكاملة لإدارة المتاجر الإلكترونية. مبنية كـ SPA تفاعلية وسريعة باستخدام Tailwind CSS و Vanilla JS، مع نظام محاكاة وحفظ بيانات ذكي عبر LocalStorage وجاهزية كاملة للربط البرمجي (API-Ready).

# 📊 لوحة تحكم متطورة للمتاجر الإلكترونية | basafer Dashboard

لوحة تحكم احترافية، عصرية، وفائقة السرعة مصممة خصيصاً لإدارة المتاجر الإلكترونية المعاصرة. تم بناء المشروع كصفحة واحدة تفاعلية (**SPA**) تقدم تجربة مستخدم سلسة واستثنائية دون الحاجة لإعادة تحميل الصفحات، مما يجعلها القالب المثالي لبدء أي مشروع متجر إلكتروني متكامل.

## ✨ ميزات لوحة التحكم (Key Features)

* **🚀 تطبيق صفحة واحدة (SPA):** تنقل فوري وسلس بين الأقسام (الرئيسية، المنتجات، الطلبات، الإعدادات) لضمان أعلى سرعة أداء.
* **📦 إدارة كاملة للمنتجات:** دعم كامل لعمليات (إضافة، تعديل، حذف، وعرض) المنتجات مع نظام حماية يضمن تحميل منتجات افتراضية مميزة عند التشغيل الأول.
* **🛒 نظام متطور لإدارة الطلبات:** جدول منظم لاستعراض الطلبات وتحديث حالاتها تفاعلياً (قيد الانتظار، مكتمل، ملغي) مع تأثير فوري على الإحصائيات.
* **📈 محرك إحصائيات ورسوم بيانية ذكي:** حساب تلقائي وفي الوقت الفعلي (Real-time) لإجمالي المبيعات، صافي الأرباح، ومعدل التحويل، مدعوماً برسم بياني تفاعلي لتحليل الأداء.
* **💾 حفظ تلقائي مستقر (State Management):** الاعتماد على الذاكرة المحلية للمتصفح `localStorage` لمزامنة البيانات وتفادي فقدانها عند تحديث الصفحة أو إغلاق المتصفح.
* **📱 تصميم مستجيب بالكامل (Responsive UI):** واجهة متوافقة بدقة عالية مع جميع الشاشات (الهواتف المحمولة، الأجهزة اللوحية، والشاشات المكتبية) بفضل هندسة Tailwind CSS.
* **🔌 جاهزية الربط البرمجي (API-Ready):** الكود مكتوب بهيكلية منظمة ومصفوفات تعتمد معايير JSON القياسية، مما يسهل ربطه مستقبلاً بأي سيرفر أو قاعدة بيانات (مثل Firebase أو SQL) في وقت قياسي.

---

## 🛠️ التقنيات المستخدمة (Tech Stack)

* **HTML5:** لبناء الهيكل البرمجي القياسي والمنظم (Semantic HTML).
* **Tailwind CSS:** لتنسيق الواجهات وبناء تصميم مرن ومستجيب ذو طابع عصري.
* **Vanilla JavaScript (ES6+):** لبرمجة المنطق الحسابي، وإدارة الحالات، والتحكم تفاعلياً في عناصر الصفحة (DOM Manipulation).
* **Lucide Icons:** لإدراج أيقونات بصرية خفيفة الوزن تضفي لمسة احترافية على الواجهة.

## 💻 طريقة التشغيل محلياً (How to Run Locally)

المشروع خفيف جداً ولا يحتاج إلى تثبيت أي حزم أو مكتبات ثقيلة، يمكنك تشغيله في ثانيتين فقط:

1. قم بعمل `clone` للمستودع أو تحميله كملف مضغوط ZIP.
2. افتح مجلد المشروع باستخدام محررك المفضل (مثل **VS Code**).
3. اضغط كليك يمين على ملف `index.html` واختر **Open with Live Server**، واستمتع بتجربة اللوحة!
